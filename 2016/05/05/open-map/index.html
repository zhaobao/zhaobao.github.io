术·北京互联网公司分布(前端)

May 5, 2016

在北京孩子，每天上下班很痛苦，挤地铁，等公交，吃早饭。。。
想做一个map，大体统计下拉勾网上互联网公司的位置，目的是为了找一个比较好方便的租房位置，以下是成果和构思过程。

可行性分析

一开始想做个静态地图，似乎效果不好，如果能取到地理位置，然后调用 map api 就好了，因此：

1 尝试拿到拉勾网互联网公司的经度纬度

  • 列表数据比较容易获取, 可以通过模拟Ajax请求搞定
  • 详情数据做了页面静态化，需要拿到positionId，然后得到页面，然后通过解析页面来获取位置信息
  • 为了防止被封杀，自己做文件缓存

2 调用百度Map Api，进行标记

  • 因为拉钩上并没有具体的经纬度信息，因此需要从地址解析到经纬度的结构
  • 需要在地图上增加标记
  • 需要在标记上增加公司名称的标签

3 增加位置房屋价格信息

  • 抓取比如我爱我家的数据
  • 如何通过经纬度得到比较密集的区域？选中区域画格子，得到每个格子定点的周围点数多少，然后排序，画圆

成果以及结论

  • 互联网公司大部分分布在朝阳区和海淀区
  • 从地铁上看，主要集中在地铁国贸、中关村以及望京附近
  • 数据供参考，没啥实际的意义，娱乐一下，一笑了之，谢
